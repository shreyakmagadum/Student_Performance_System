# Student_Performance_System
A Python-based CLI application for managing and analyzing student academic performance using OOP and CSV file handling.

## Overview
The **Student Performance Analysis System** is a Python-based command-line application designed to manage and analyze student academic performance.

This project demonstrates the use of **Object-Oriented Programming (OOP)**, **CSV file handling**, and **basic statistical analysis** using pure Python.

## Features
- Add new student records
- Display all student information
- Search student by ID
- Identify the class topper
- Find subject-wise highest marks
- Calculate class average
- Compute subject-wise averages
- Calculate standard deviation of marks
- Display grade distribution
- Load student data from CSV
- Save student data to CSV

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- CSV File Handling
- Command Line Interface (CLI)
